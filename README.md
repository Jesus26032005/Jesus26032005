## 🚀 Proyectos Destacados

### ☕ [BitCafe API](https://github.com/Jesus26032005/api-test)
Backend robusto para la gestión integral de una cafetería. Maneja autenticación, usuarios, inventario, carritos de compra, pedidos y pagos.
* **Stack:** FastAPI, SQLModel/SQLAlchemy, JWT (JOSE), Passlib (Argon2).
* **Features:** Integración de pagos con Mercado Pago y manejo de imágenes estáticas.

### 🌐 Union Zalor
Sitio web interactivo para una comunidad, con navegación fluida, secciones públicas y un panel de administración simulado (vía `localStorage`).
* **Stack:** React 18, React Router, Vite, Tailwind CSS, Framer Motion.
* *(En desarrollo / Repo privado por el momento)*

### 🖼️ Procesamiento de Imágenes
Aplicación de escritorio estructurada bajo el patrón MVC para operaciones visuales: aplicación de filtros, segmentación, análisis de histogramas y más.
* **Stack:** Python, Tkinter (ttkbootstrap), OpenCV, NumPy, Matplotlib, SciPy.
* *(Proyecto local)*

### 📚 OrientaTEC
Sistema web integral para gestión de información apoyado por bases de datos relacionales.
* **Stack:** PHP, JavaScript, SQL.
* *(Proyecto local: Nota de desarrollo sobre recuperación de scripts DB).*
